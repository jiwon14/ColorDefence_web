특성 상세 정보 (메인 특성)

게임에 큰 방향성을 정하는 메인 특성을 확인해보세요!

태극

동일한 직업에서 빨강, 파랑 유닛만 존재하면 그 유닛들은 강화됩니다.

기사 : 50 -> 200
궁수 : 250 -> 1450
창병 : 4000 -> 19,000
마법사 : 25,000 -> 135,000

컬러 마스터

유닛 뽑기 시 초록,보라,주황 기사가 등장할 수 있습니다.

도박사

화면 중앙 아래 생기는 버튼으로 스택을 쌓은 후 유닛을 뽑을 수 있습니다.
스택은 골드로 구매하거나, 라운드가 끝날 때 마다 올라갑니다.
유닛을 뽑을수록 더 좋은 유닛이 등장할 수 있습니다.

네크로맨서

적 유닛을 특정 수 만큼 처치하면 보라 기사를 소환합니다.
최대 유닛을 초과해서 유닛을 획득가능합니다.

마창사

창병이 강화된 창을 날립니다.
강화된 창은 2배의 피해량을 입힙니다.

VIP

상점에서 특정 개수만큼의 상품을 구매하면 VIP 상점이 열립니다.
VIP상점에서는 유닛과 더 강력한 효과의 강화효과를 구매할 수 있습니다.

백의 결속

하얀 유닛이 강화되고 지속시간이 4배 증가합니다.(30초->120초)
5라운드마다 룬을 2개 획득합니다.

기사 : 500 -> 600
궁수 : 3000 -> 6000
창병 : 50,000 -> 111,000
마법사 : 1,000,000 -> 6,000,000

흑의 결속

검은 유닛이 강해집니다.

기사 : 1200 -> 19,200
궁수 : 15,000 -> 915,000
창병 : 1,000,000 -> 6,000,000
마법사 : 10,000,000 -> 16,000,000

황금빛 기사

모든 노란 유닛이 강화되며 기사가 조합에 사용될 시 골드가 증가합니다.
유닛 데미지 증가량 : 10%
기사 조합시 골드 : 1 -> 3

메테오

빨간 기사를 소환하거나 조합에 사용되면 메테오를 떨어뜨립니다.
메테오의 데미지는 조합에 사용할 때 마다 증가하는 스택에 비례합니다.

기본 데미지: 1000
스턴 시간: 스택 X 0.1
스택 당 추가 데미지 : 2500

빨간 기사 10마리 당 스택 1이 증가합니다.

썬콜

슬로우가 적용된 적에게 다시 슬로우를 적용시키면 번개로 적을 공격합니다.
추가로 슬로우(이동속도 감소 효과) 시간이 5배 증가합니다.
